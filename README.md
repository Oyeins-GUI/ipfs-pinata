# Run locally

```bash
git clone https://github.com/Oyeins-GUI/ipfs-pinata.git
cd ipfs-pinata
npm install
# or
yarn add
npm run dev
```

## API key, JWT & Gateway URL

-  Visit [pinata](https://app.pinata.cloud/developers/api-keys) to get your tokens
-  ```bash
   VITE_PINATA_JWT=
   VITE_GATEWAY_URL=
   ```
